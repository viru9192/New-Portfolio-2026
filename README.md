# Viranchi More — Portfolio

Personal portfolio website built with React, Tailwind CSS, and Framer Motion.

**Live site:** https://viru9192.github.io/New-Portfolio-2026

## Tech Stack

- **React 18** — component-based UI
- **Tailwind CSS** — utility-first styling
- **Framer Motion** — animations and transitions
- **Recharts** — data visualization in the dashboard
- **Vite** — build tool and dev server

## Sections

- Hero with typewriter animation
- About
- Skills
- Projects
- Experience
- GitHub activity
- Interactive dashboard
- Contact

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deploy

```bash
npm run deploy
```

Deploys to GitHub Pages via the `gh-pages` package.
